# Lyzr Brand Do's and Don'ts

## Logo — Never Do This

| Violation | Description |
|-----------|-------------|
| **Stack** | Never place wordmark below the logomark — always horizontal lockup |
| **Rotate** | Never rotate the logo or any part of it at any angle |
| **Reorder** | Never swap or reorder symbol vs wordmark position |
| **Resize parts** | Never change the size of one part relative to the other (symbol vs "lyzr" text) |
| **Outline** | Never use an outlined/stroked version of the logo |
| **Distort** | Never stretch, squeeze, or warp the logo in any dimension |
| **Opacity change** | Never reduce logo opacity — always full opacity |
| **Low contrast BG** | Never place logo on a background where it can't be clearly read |
| **Add elements** | Never add text (e.g., "lyzr Marketplace"), icons, or decorations to the logo |
| **Busy BG** | Never place logo on a busy photograph or complex pattern |
| **Wrong color** | Never recolor the logo outside of approved versions (dark/light variants only) |
| **Too small** | Never use full logo below 100px wide, never use logomark below 24px wide |
| **Encroachment** | Never let other elements enter the ½-height clear space zone |

## Logo — Always Do This

- Use the correct logo variant for the background (dark logo on light, light logo on dark)
- Maintain ½ logo height clear space on all four sides — no exceptions
- On social media: logo at top, minimum 160px wide
- On thumbnails: logo visible, not competing with main title

## Colors — Never Do This

| Violation | Why |
|-----------|-----|
| Use pure white `#FFFFFF` as primary background | Use White Amber `#F3EFEA` instead |
| Use blue as a "tech" primary | Lyzr's identity is Ferra, not blue |
| Use vibrant/neon accent colors | All accents must be matte; no oversaturation |
| Use washed-out/pastel versions of accents | Also not allowed — matte ≠ washed out |
| Make accent colors dominant | They are 10% of the palette, not 60% |
| Use unapproved colors | Stick to the palette; new colors require brand review |
| Place Lyzr Black text on Lyzr Ferra without checking contrast | Always verify AA compliance |
| Use colorful gradients | Only opacity-based gradients within the tonal scale |

## Colors — Always Do This

- Start every design with the 60-30-10 rule as a baseline
- Test every text/background pair for WCAG AA minimum
- Use the tonal scale (not random tints) when you need intermediate values
- Constrain accent use to infographics, alerts, CTAs, and emphasis
- Maintain matte finish throughout — this is what makes the brand feel premium

## Typography — Never Do This

| Violation | Why |
|-----------|-----|
| Use Playfair Display Light or Regular for headings | Too thin, kills legibility and brand impact |
| Mix multiple serif fonts | Playfair Display is the only brand serif |
| Use a random sans-serif for headings | Must be Playfair Display |
| Go below medium weight on Playfair for any visible heading | Minimum is Medium (500) |
| Use Comic Sans, Papyrus, or decorative fonts | Obviously off-brand |
| Apply all-caps to Playfair Display | Not part of the brand language |
| Use tight line spacing below 115% | Compromises readability |
| Use loose line spacing above 150% for body | Too airy, breaks rhythm |

## Typography — Always Do This

- Playfair Display for every heading/emphasis — no exceptions
- Noto Sans (or Inter fallback) for all body, subheadings, UI text
- Respect minimum font sizes per medium (see layout-rules.md)
- Weight hierarchy: headings boldest → body regular → captions lightest

## Layout — Never Do This

| Violation | Why |
|-----------|-----|
| Ignore padding minimums | Cramped layouts look unprofessional |
| Put too much on one slide/creative | Complexity kills the message |
| Mix alignment styles within a piece | Breaks visual rhythm |
| Use drop shadows | Not part of Lyzr's flat/layered aesthetic |
| Use 3D effects | Same — flat, matte, layered is the brand |
| Right-align body text blocks | Hurts readability for paragraphs |
| Use busy photo backgrounds behind logo | Logo always on clean surface |
| Place logo below minimum size for medium | See logo rules |

## Layout — Always Do This

- Apply padding minimums strictly (70px social, 90px carousel, 100/160px YouTube)
- Keep layouts clean and uncluttered — split content before cramming
- Use Elevate-style layered brand elements for decorative depth (not random clip art)
- Logo position consistent throughout a multi-piece project
- One clear visual hierarchy: what's most important draws the eye first

## Brand Feel — Quick Gut Check

Ask yourself: does this look like it belongs in a premium, sophisticated AI infrastructure brand?

**Looks right:**
- Warm earth tones, matte surfaces
- Restrained use of the Ferra accent
- Clean Playfair Display headlines
- Layered diagonal/circular depth elements in Ferra family tones
- Plenty of White Amber breathing room

**Looks wrong:**
- Bright blues, purples, greens dominating
- Loud gradients
- Thin, wispy fonts
- Cluttered with multiple accent colors
- Drop shadows, glows, or 3D effects
- Feels like a generic SaaS company — it should feel like a premium infrastructure brand

---

## Competitive Differentiation Note

Lyzr deliberately chose warm earth tones (Ferra) over the blue/purple tech aesthetic.
This is intentional brand positioning — do not "fix" it toward blue because it feels more "techy."
The warmth IS the differentiator. Protect it.
